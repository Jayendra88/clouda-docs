The opportunity

🚀

Launch a new brand fast

Standing up a new brand website becomes a repeatable setup — its own web address, its own identity — instead of a fresh build every time.

🎛️

Manage everything in one place

Your team logs in once, chooses which brand they’re working on, and edits that brand’s pages — all from a single content system.

🎨

Every brand looks unique

Each brand gets its own theme — its own colours, fonts and logo — even though they share the same underlying building blocks.

🧩

A tidy, shared toolkit

How it works

The big picture

One content system and one shared foundation feed many independent brand websites. Each brand’s content is kept strictly separate, and each site is published on its own domain with its own theme.

Peace of mind

Safe by design

Because many brands share one system, keeping their data and access separate is the top priority. This separation is built into the core of the system, so it’s applied automatically everywhere — rather than depending on each page to remember.

✓
Each brand’s content stays on its own siteEvery request is automatically limited to the brand being worked on. The separation lives in the core of the system, not in each individual screen.
✓
Editors only touch their assigned brandsAccess is granted per brand. Someone can be an editor on one brand and have no access to another.
✓
Separate, independent websitesEach brand’s website runs on its own, so a change or issue on one site doesn’t affect the others.
✓
Nothing breaks during the switch-overThe current website keeps working exactly as it does today throughout the rollout; multi-brand capability is added underneath it, step by step.
✓
A more robust system overallAlong the way we’ll also tidy up a few configuration items we noticed while reviewing the system, making it sturdier for everyone.
The plan

How we’ll roll it out

We start with the essential foundation and prove the whole idea end-to-end by taking one real second brand fully live. Enhancements follow once that’s working.

Step 1

Lay the foundation

Introduce the brand concept into the data, with the safeguards that keep every brand’s content separate — all without changing how the current site behaves.

Step 2

Make the public site brand-aware

The website learns which brand it is serving, pulling that brand’s content, settings and analytics (its own visitor stats and marketing tags).

Step 3

Add per-brand theming

A theming layer so each brand gets its own look. Colours, fonts and logo apply to a new brand right away; finer styling of individual components is refined over time.

Step 4

Brand picker & permissions in the editor

The content system gets the brand switcher, per-brand access, and built-in checks that prevent anyone from editing the wrong brand by mistake.

Step 5 🎯

Take a second brand live

The milestone: a real second brand, live on its own domain with its own theme — proving the whole process works, from building the pages to visitors seeing the live, branded site.

Later

Platform enhancements

A searchable component catalogue, admin screens to manage brands and themes with live preview, the full component styling refresh, and per-brand backups.

🎯 Key milestone: one real second brand fully live on its own domain with its own theme — the point at which the multi-brand platform has proven itself.
Key choices

Why we recommend this approach

Four decisions shape the plan. In each case we’ve chosen the option that gets you to value fastest and safest, while leaving room to grow.

DecisionOur recommendationWhy
How brand data is kept separateOne shared store, tagged by brandStrong isolation with the simplest, lowest-cost setup — and a clear path to fully separate storage later for any brand that needs it.
How each site is publishedEach brand runs on its own siteMatches “separate sites on separate domains,” gives the strongest isolation, and keeps each site simple and independent.
How brand styling is deliveredFoundation first, then refineNew brands get a distinct look immediately; the deeper component refresh happens gradually with low risk.
How ambitious to be at firstProve it with one second brandDelivers real value quickly and de-risks the approach before investing in the broader platform tooling.
